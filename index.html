Git Config

git config –list
Muestra la configuración de variables globales de Git.

git clone
Realiza una copia exacta de un repositorio en la computadora local.

git init Configura un directorio local para iniciar el CVS sobre los archivos del mismo.

git remote add origin https://github.com/< user >/< Proyecto >.git
Establece la dirección a donde se va a subir el código.

git add < archivo >
Agrega a un archivo en estado modified a estado staged.

git commit
Etiqueta con una descripción los archivos modificados y mantenerlos almacenados localmente.

git push
Realiza el commit subiendo los cambios físicamente al servidor.

git pull
Realiza una actualización desde el servidor a la computadora local, con la ultima versión de los archivos.

git status
Muestra las modificaciones realizadas en un archivo.

Comandos Branch:

git branch
Ver en que branch estas trabajando.

git branch < nombre del branch >
Crear un nuevo branch y checkout para cambiarme al nuevo branch.

git checkout –b < branch >
Bajar un branch.
